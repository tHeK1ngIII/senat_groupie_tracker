# s-nat_groupie_tracker

Dans le cadre de notre apprentissage de l'informatique nous avons pour projet de faire un site avec une API.
Nous avons décider d'utiliser celle de clash royal.
J'ai une petite surprise pour toi Kantounet.

11/12/25

Nous avons créer notre compte sur le site devlopper de clash royal et nous avons mis la clé d'autorisation d'accés à l'API en public car Supercelle impose obligatoirement une liste d'IP autorisées pour des raison de sécurité.
Actuellement nous testons sur Insomnia pour voir si ca marche. Dans le cas contraire si cela se revelle infructueux nous serons obligé de prévoire une back end.
